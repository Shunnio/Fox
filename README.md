
## Tính năng nổi bật
* **Robust Controller:** Hệ thống điều khiển nhân vật mượt mà (Di chuyển, Nhảy).
* **Tilemap System:** Môi trường được xây dựng tối ưu bằng Unity Tilemap.
* **Level Design:** Cấu trúc màn chơi đa dạng với hệ thống cạm bẫy và vật phẩm thu thập.
* **UI/UX:** Giao diện Menu chính, Menu tạm dừng và hệ thống điểm số hoàn thiện.
